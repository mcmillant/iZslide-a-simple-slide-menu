iZslide - demo

by mcmillant

This is a simple example about slide-menu. when you .click button, it will .addClass to show menu, and it will add a "fog" over the website to alert user that they are working with slide menu.
When.click once again in close button or the fog it will .removeClass.
In here i used Jquery so .toggleClass is pretty good - Add or remove one or more classes from each element in the set of matched elements, depending on either the class's presence or the value of the switch argument.
